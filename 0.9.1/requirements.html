Requirements¶

For the IBM Z HMC collection, the managed node is the control node, i.e. the playbook has its hosts and connection properties set accordingly:

- hosts: localhost
  connection: local

The location of the HMC is defined with input parameters to the modules.

Control node¶

Besides having Ansible and the IBM Z HMC collection installed, there are no additional requirements for the control node.
